HTML(HyperText Markup Language)은 웹페이지의 실질적인 내용을 위한 언어이다. 내용과 함께 태그를 이용하여 웹페이지의 구조를 나타내는 문서를 작성한다. 더 자세한 내용 보러가기!



내가 보려고 만든 HTML 태그 목록

<h1>~<h6>: 제목
<strong>: 굵은글씨로 강조
<u>: 밑줄
<br>: 줄바꿈
<p>: 문단
<img>: 이미지
<li>: 목차(list)
<ul>: 순서가 없는 목차(unordered list)
<ol>: 순서가 있는 목차(ordered list)
<title>: 페이지 제목을 나타내는 태그
<meta>: 웹페이지 저장방식 ex)UTF-8
<head>: 본문을 설명하는 부분. 페이지 제목, meta태그 등이 들어간다.
<body>: 본문을 나타냄
<html>: head와 body를 감싸는 태그
<a>: 링크
<div>: 디자인을 위해 감싸는 태그
<hr>: 단락을 구분하는 가로선

사용빈도가 높은 태그 보러가기.
